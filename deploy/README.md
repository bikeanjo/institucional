# Deploy - Quick Reference

## Quick Start

```bash
# Stage (local)
./start.sh -stage

# Production (server)
./start.sh -prod
```

## What's Here

- `start.sh` - Deployment script
- `docker-compose-bikeanjo-institucional.yml` - Container config
- `Dockerfile.frontend` - React + Nginx
- `Dockerfile.api` - Node.js API
- `nginx.conf` / `nginx-host.conf` - Web server configs
